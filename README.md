# PROJECT 3: RANDOM PASSWORD GENERATOR
## Overview
The Random Password Generator is a simple Python project that helps users generate secure and random passwords.The program asks the user to enter the required password length and then creates a strong password using letters and numbers.This project is useful for understanding basic security concepts and Python programming.
## Objective
The main objective of this project is to learn Python basics such as string handling,loops,random module usage,and user input.It also helps in understanding how passwords can be generated programmatically for better security.
## Tools Used
- Python
- VS Code
- Command Prompt
## Conclusion
By completing this project,I learned how to use Python modules like `random` and `string` to generate secure passwords.This project improved my understanding of string manipulation,loops,and user input handling.It also helped me improve my coding and logical thinking skills by creating a simple security-based application.
